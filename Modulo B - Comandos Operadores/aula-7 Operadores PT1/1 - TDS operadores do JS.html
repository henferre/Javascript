Operadores Aritméticos em JS:

MAIS: 5+2 = 7
MENOS: 5-2 = 3
MULTIPLICAÇÃO: 5*2 = 10
DIVISÃO REAL: 5/2 = 2.5
RESTO DIVISÃO: 5%2 = 1
POTENCIA : 5**2 = 25

Ordem de Procedência em JS:

1º ()
2º **
3º * ou / ou % Quem aparecer primeiro.
4º + e - Quem aparecer primeiro.

Atribuição Simples em JS:

var a = 5 + 3 == 8
var b = a % 5 == 3
var c = 5 * b ** 2 == 45
var d = 10 - a / 2 == 6
var e = 6 * 2 / d == 2
var f = b % e + 4 / e == 3

Auto-atribuições em JS:

var n = 3
n = n + 4 == 7
n = n - 5 == 2

Auto-atribuições simplificada em JS:

var n = 3
n += 4 == 7
n -= 3 == 0

Incremento em JS:

var x = 5
Ao inves de vc usar x += 1 == 6 vc pode usar x ++
Dessa forma daria o mesmo resultado.
