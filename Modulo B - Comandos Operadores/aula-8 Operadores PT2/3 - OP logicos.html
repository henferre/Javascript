Operadores logicos em JS:

! - Negação:

Esse operador é mais basico
ou a resposta é sim ou não

True --> False
False --> True

&& - Conjunção:

Esse operador só se satisfaz quando
ambos são iguais, igual and Python.

true && true? --> True
true && false? --> False
false && true? --> False
false && false? --> True

|| - Disjunção:

Esse operador já é o contrario
da conjunção, igual or no Python.

true || true? --> True
true || false? --> True
false || true? --> True
false || false? --> False
